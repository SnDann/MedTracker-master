# Roteiro de Testes Manuais - MedTracker

## 1. Autenticação
- [ ] Registrar novo usuário
- [ ] Fazer login com usuário existente
- [ ] Fazer logout

## 2. Medicamentos
- [ ] Adicionar novo medicamento (nome, dosagem, dias, horários, observações)
- [ ] Editar medicamento existente
- [ ] Excluir medicamento
- [ ] Marcar medicamento como tomado
- [ ] Verificar se medicamentos aparecem corretamente na agenda

## 3. Notificações
- [ ] Permitir notificações no dispositivo
- [ ] Verificar se lembretes são disparados nos horários agendados
- [ ] Editar/remover medicamento e garantir que notificações são atualizadas/canceladas

## 4. Exportação/Importação
- [ ] Exportar dados de medicamentos (gerar arquivo JSON)
- [ ] Importar dados de medicamentos (usar arquivo exportado)
- [ ] Tentar importar arquivo inválido e verificar mensagem de erro

## 5. Câmera e OCR
- [ ] Capturar imagem de receita
- [ ] Verificar se o texto é extraído automaticamente
- [ ] Salvar prescrição com texto extraído
- [ ] Verificar se prescrição aparece no histórico

## 6. Configurações e Privacidade
- [ ] Acessar tela de configurações
- [ ] Visualizar política de privacidade
- [ ] Testar exportação/exclusão de dados

## 7. Fluxo Geral
- [ ] Navegar entre todas as abas sem erros
- [ ] Testar o app em modo offline (verificar mensagens e comportamento)

---
Preencha este roteiro durante os testes e reporte qualquer erro ou comportamento inesperado.
