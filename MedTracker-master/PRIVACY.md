# Política de Privacidade - MedTracker

Esta política de privacidade descreve como o MedTracker coleta, usa, armazena e protege suas informações.

## 1. Coleta de Dados
- Coletamos apenas os dados necessários para o funcionamento do app: nome, e-mail, medicamentos, horários, imagens de receitas e texto extraído (OCR).
- Não coletamos dados sensíveis além dos informados pelo próprio usuário.

## 2. Uso dos Dados
- Os dados são utilizados exclusivamente para fornecer as funcionalidades do app, como lembretes, histórico e exportação/importação.
- O texto extraído de receitas é salvo apenas na sua conta e não é compartilhado.

## 3. Compartilhamento de Dados
- Não compartilhamos seus dados pessoais ou de saúde com terceiros.
- Você pode exportar seus dados a qualquer momento em formato JSON.

## 4. Armazenamento e Segurança
- Os dados são armazenados de forma segura no Supabase, com autenticação e criptografia.
- Imagens e textos extraídos ficam acessíveis apenas ao usuário autenticado.

## 5. Direitos do Usuário
- Você pode acessar, exportar ou excluir seus dados a qualquer momento.
- Para solicitar exclusão total, entre em contato pelo suporte do app.

## 6. Alterações na Política
- Esta política pode ser atualizada. Notificaremos os usuários em caso de mudanças significativas.

## 7. Contato
- Para dúvidas ou solicitações, entre em contato pelo e-mail: suporte@medtracker.com
